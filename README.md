## ۲. آموزش کامل کدبیس

### 📦 مدل‌های دیتابیس (`models.py`)

این فایل قلب مدیریت چند-مستاجری سیستم است. ساختار دیتابیس بر اساس ۶ مدل اصلی بنا شده است:

1. **Feature (قابلیت):** تعریف ویژگی‌هایی که می‌توان به پلن‌ها اختصاص داد (مانند `price_manager` یا تغییر قیمت گروهی).
2. **Plan (پلن اشتراکی):** انواع سطوح اشتراک (پایه، حرفه‌ای، پیشرفته) که هر کدام به چند Feature متصل هستند.
3. **Store (مستاجر/فروشگاه):** حیاتی‌ترین مدل سیستم. هر فروشگاه ثبت‌نام شده یک مستاجر مجزا با کلیدهای API اختصاصی ووکامرس و هوش مصنوعی خودش است.
4. **UserProfile (پروفایل کاربر):** کاربران جنگو را به یک `Store` متصل کرده و نقش آن‌ها را (ادمین یا اپراتور) تعیین می‌کند.
5. **ProductLog (گزارش محصولات):** تاریخچه محصولات تولید شده توسط هوش مصنوعی را برای محاسبه دقیق سهمیه‌های ماهانه ذخیره می‌کند.
6. **InternalLink (لینک‌های داخلی):** ساختار لینک‌سازی داخلی هر فروشگاه برای استفاده هوشمندانه در متن‌های تولید شده توسط AI.

## ۴. نصب Docker Engine — گام به گام

> 📌 **داکر چیست؟** داکر پلتفرمی است که برنامه شما را به همراه تمامی نیازمندی‌هایش (پایتون، دیتابیس، پکیج‌ها) در یک محیط ایزوله و مستقل به نام «کانتینر» بسته‌بندی می‌کند تا بدون مشکلِ تغییرات محیطی، روی هر سیستمی دقیقاً یکسان اجرا شود.

### 🐧 نصب روی توزیع‌های لینوکس (Ubuntu / Debian)

۱. حذف پکیج‌های قدیمی و احتمالی:

```bash
sudo apt-get remove docker docker-engine docker.io containerd runc

```

۲. بروزرسانی مخازن و نصب ابزارهای پیش‌نیاز:

```bash
sudo apt-get update
sudo apt-get install -y ca-certificates curl gnupg lsb-release

```

۳. ساخت پوشه کلیدها و دریافت کلید GPG رسمی داکر:

```bash
sudo mkdir -m 0755 -p /etc/apt/keyrings
curl -fsSL [https://download.docker.com/linux/ubuntu/gpg](https://download.docker.com/linux/ubuntu/gpg) | sudo gpg --dearmor -o /etc/apt/keyrings/docker.gpg

```

۴. تنظیم و ثبت ریپازیتوری رسمی در سیستم:

```bash
echo \
  "deb [arch=$(dpkg --print-architecture) signed-by=/etc/apt/keyrings/docker.gpg] [https://download.docker.com/linux/ubuntu](https://download.docker.com/linux/ubuntu) \
  $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null

```

۵. نصب نسخه نهایی Docker Engine و متعلقات داکر کامپوز:

```bash
sudo apt-get update
sudo apt-get install -y docker-ce docker-ce-cli containerd.io docker-buildx-plugin docker-compose-plugin

```

> ⚙️ **نکته کاربران ویندوز و مک:** نیازی به اجرای دستورات ترمینالی بالا ندارید. کافیست نرم‌افزار **Docker Desktop** را از وب‌سایت رسمی داکر دانلود و نصب کنید.
